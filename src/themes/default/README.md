## 架构说明

默认主题，包含落地页区块、布局和页面模板。
通过 getThemeBlock/getThemePage/getThemeLayout 动态加载。

## 文件索引

| 目录/文件 | 功能描述 |
|-----------|----------|
| blocks/ | 落地页区块（hero、features、pricing、faq 等） |
| layouts/ | 页面布局（landing） |
| pages/ | 页面模板（static-page、dynamic-page） |
