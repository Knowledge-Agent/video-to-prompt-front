## 架构说明

核心基础设施层，提供认证、数据库、国际化、权限控制等底层能力。
所有模块通过 index.ts 导出统一接口，供上层调用。

## 文件索引

| 目录/文件 | 功能描述 |
|-----------|----------|
| auth/ | better-auth 认证配置和客户端 |
| db/ | Drizzle ORM 多数据库连接（postgres/mysql/sqlite） |
| docs/ | fumadocs 文档源配置和目录生成 |
| i18n/ | next-intl 国际化路由和请求配置 |
| rbac/ | 角色权限控制系统 |
| theme/ | 主题加载器和 Provider |
