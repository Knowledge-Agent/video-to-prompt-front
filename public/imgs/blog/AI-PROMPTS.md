## 🎨 生成技巧

### Midjourney 参数说明
- `--ar 16:9`: 设置宽高比为 16:9
- `--v 6`: 使用 Midjourney v6 模型
- `--niji 6`: 使用动漫风格模型（仅用于动漫图片）
- `--q 2`: 提高质量（可选）
- `--stylize 500`: 调整风格化程度

### DALL-E 技巧
- 使用 "Create a..." 开头更容易理解
- 明确指定风格："modern tech illustration", "professional photography"
- 描述颜色方案："blue and purple gradient"
- 强调质量："high detail", "professional quality"

### Stable Diffusion 技巧
- 使用逗号分隔关键词
- 添加质量词："8k", "high quality", "professional"
- 指定风格："photorealistic", "illustration", "digital art"
- 使用负面提示词（Negative Prompt）排除不想要的元素：
  ```
  Negative: blurry, low quality, watermark, text, signature
  ```

## 📝 批量生成建议

### 1. 使用一致的风格
所有图片应保持一致的视觉风格：
- **配色方案**: 蓝色、紫色、白色为主
- **设计风格**: 现代、简洁、专业
- **字体**: 无衬线字体（如果包含文字）

### 2. 尺寸规范
- **Hero 图片**: 1200x630px (社交媒体标准)
- **对比图片**: 1600x900px (16:9)
- **技术图表**: 1200x800px (3:2)

### 3. 后期处理
生成后建议进行：
- 压缩优化（TinyPNG, Squoosh）
- 添加文字标注（Figma, Canva）
- 统一色调（Photoshop, GIMP）

## 🚀 快速生成流程

### 方案 A: Midjourney（推荐）
1. 复制上述提示词到 Midjourney
2. 使用 `/imagine` 命令
3. 选择最佳结果并 Upscale
4. 下载并重命名文件

### 方案 B: DALL-E
1. 访问 ChatGPT 或 DALL-E 网站
2. 粘贴提示词
3. 生成并下载
4. 可能需要多次生成选择最佳

### 方案 C: Stable Diffusion（本地）
1. 使用 Automatic1111 WebUI
2. 输入提示词和负面提示词
3. 设置参数：
   - Steps: 30-50
   - CFG Scale: 7-9
   - Sampler: DPM++ 2M Karras
4. 生成并选择最佳结果
