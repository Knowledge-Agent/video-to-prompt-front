# 提升提示词质量的镜头语言框架

很多提示词失败，不是因为“内容不对”，而是因为只写了“拍什么”，没写“怎么拍”。

![很多提示词失败，不是因为“内容不对”，而是因为只写了“拍什么”，没写“怎么拍”。](/imgs/blog/tavily/video-prompt-shot-language-framework.zh-2.png)

高质量提示词应同时覆盖：

![高质量提示词应同时覆盖：](/imgs/blog/tavily/video-prompt-shot-language-framework.zh-3.png)

- 构图（远景、中景、近景）
- 机位运动（平移、推进、手持）
- 镜头意图（压缩感、景深、形变）
- 节奏（慢推进 / 快切）

## 可直接套用的镜头句式

建议使用：

![建议使用：](/imgs/blog/tavily/video-prompt-shot-language-framework.zh-4.png)

> 主体 + 场景 + 相机动作 + 镜头/景深 + 光线 + 情绪 + 时间推进

示例：

![示例：](/imgs/blog/tavily/video-prompt-shot-language-framework.zh-5.png)

> 黄昏雨巷中，跑者冲入画面，手持中景跟拍，50mm 质感的浅景深，霓虹侧逆光与地面积水反射，紧张电影氛围，镜头缓慢推进并最终收于近景。

## 常见问题

- 只写物体，不写运动
- 缺少景深和镜头信息，画面容易“平”
- 风格标签互相冲突
- 缺少时间推进线索

## 总结

镜头语言是“参考视频”到“可控生成”之间的关键桥梁。写清镜头逻辑，结果可控性会显著提高。
