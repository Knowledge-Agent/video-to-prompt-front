# YouTube Shorts Hook Prompt Templates (First 3 Seconds)

For Shorts, the first 3 seconds decide retention. These templates are designed for teams using **video prompts** to test hooks quickly and systematically.

![For Shorts, the first 3 seconds decide retention. These templates are designed f](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-2.jpg)

## Hook formula

`Pattern Interrupt + Subject Action + Camera Move + Visual Payoff`

![`Pattern Interrupt + Subject Action + Camera Move + Visual Payoff`](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-3.jpg)

## 5 reusable hook templates

### Template 1: Motion snap

A fast directional movement that resolves into a clear subject close-up.

### Template 2: Reveal switch

Start obscured, then reveal product/person in one clean motion.

![Start obscured, then reveal product/person in one clean motion.](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-4.jpg)

### Template 3: Textless question shot

No text overlay. Visual contradiction creates curiosity.

![No text overlay. Visual contradiction creates curiosity.](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-5.jpg)

### Template 4: Before/after flash

One-frame contrast cue then immediate hero shot.

![One-frame contrast cue then immediate hero shot.](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-6.jpg)

### Template 5: Human reaction lead

Emotion-first face shot, then context reveal.

![Emotion-first face shot, then context reveal.](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-7.webp)

## Prompt example

```text
A creator opens frame with a sudden side-step into close-up, handheld medium tracking shot, shallow depth and high edge contrast, neon practicals in background, urgent but clean social-video mood, immediate reveal by second two.
```

## FAQ

### Should hooks always be fast?

Not always. Slow hooks can work if tension is clear from frame one.

![Not always. Slow hooks can work if tension is clear from frame one.](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-8.jpg)

### How many hook variants should I test?

Start with 5-8 structured variants per concept.

![Start with 5-8 structured variants per concept.](/imgs/blog/tavily/youtube-shorts-hook-prompt-templates-9.jpg)
