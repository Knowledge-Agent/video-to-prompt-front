# How to Get Prompt from Video

If you want to **get prompt from video** reliably, do not start by writing prompts from scratch. Start with one reference clip, break it into scene logic, then convert that logic into a structured prompt package. This approach works better than guessing because it preserves motion, framing, and pacing. In practice, a good **video to prompt generator** should output master prompt, short prompt, negative prompt, keyword tags, and shot list in one run.

![If you want to **get prompt from video** reliably, do not start by writing promp](/imgs/blog/tavily/how-to-get-prompt-from-video-2.png)

## Quick answer first

The fastest way to **generate prompt from video** is:

![The fastest way to **generate prompt from video** is:](/imgs/blog/tavily/how-to-get-prompt-from-video-3.png)

1. Pick one short reference clip (8-30 seconds).
2. Split it into 3-8 shots.
3. Describe each shot with camera action, subject, and lighting.
4. Merge shot notes into one master prompt.
5. Add a short prompt and negative prompt.
6. Save keywords for reuse in your prompt library.

This gives you controllable **video generation prompts** instead of random outputs.

## The 6-step workflow in detail

### Step 1: Choose a reference clip with clear intent

Pick a clip with one clear purpose: hook, product close-up, or transition. Avoid noisy clips with too many scenes.

![Pick a clip with one clear purpose: hook, product close-up, or transition. Avoid](/imgs/blog/tavily/how-to-get-prompt-from-video-8.jpg)

### Step 2: Break the clip into shot units

Mark shot boundaries and label each unit with:

![Mark shot boundaries and label each unit with:](/imgs/blog/tavily/how-to-get-prompt-from-video-9.jpg)

- subject
- movement
- framing
- lighting
- mood

This is where most **video prompts AI** quality is decided.

![This is where most **video prompts AI** quality is decided.](/imgs/blog/tavily/how-to-get-prompt-from-video-10.jpg)

### Step 3: Write shot-level prompt lines

For each shot, use one sentence pattern:

![For each shot, use one sentence pattern:](/imgs/blog/tavily/how-to-get-prompt-from-video-11.jpg)

`Subject + Setting + Camera Action + Lens/Depth + Lighting + Mood + Timing`

![`Subject + Setting + Camera Action + Lens/Depth + Lighting + Mood + Timing`](/imgs/blog/tavily/how-to-get-prompt-from-video-12.jpg)

### Step 4: Merge into one master prompt

Combine all shot lines into one coherent paragraph. Keep temporal order explicit, such as "starts wide, then pushes in, then ends in close-up."

![Combine all shot lines into one coherent paragraph. Keep temporal order explicit](/imgs/blog/tavily/how-to-get-prompt-from-video-13.png)

### Step 5: Add short and negative prompts

- short prompt: compressed version for fast iteration
- negative prompt: what to avoid (flicker, warped faces, inconsistent hands, noisy background)

### Step 6: Save keyword tags for retrieval

Store 5-12 tags for each package, such as "neon alley", "handheld tracking", "slow push-in". This makes future reuse much faster.

## Common mistakes

- describing objects but not camera motion
- mixing conflicting style directions in one line
- skipping negative prompts
- using too many vague tags ("cool", "cinematic", "beautiful")
- no internal naming standard for prompt packages
