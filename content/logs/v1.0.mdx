## First public workflow release

Video to Prompt v1.0 introduced the first usable workflow for turning one reference video into a structured prompt package.

### Included in v1.0

- Reference video upload and analysis
- Scene summary generation
- Master prompt and short prompt output
- Negative prompt generation
- Keyword extraction
- Shot list generation

### Why this mattered

The goal of v1.0 was not to ship a generic AI tool. It was to validate a repeatable workflow for creators and teams who constantly translate reference footage into prompts.

### Result

Users could go from one raw clip to a reusable prompt package in a single pass.
