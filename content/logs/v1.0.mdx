Video to Prompt 1.0 is live.

This first release focuses on one core workflow:

1. Upload a reference video
2. Analyze and decompose scene language
3. Copy a structured prompt package for imitation shooting

## What is included

- Prompt package generation (master / short / negative)
- Keyword extraction for indexing
- Shot list generation for execution planning
- Chinese and English output support
- Credits-based usage model

## Why v1.0 matters

Instead of writing prompts from scratch, teams can now convert real reference clips into reusable prompt assets.

This makes prompt quality more stable and collaboration more predictable.

## Next direction

Future updates will focus on:

- Better scene segmentation quality
- Faster analysis speed
- Workflow features for team-scale prompt operations
