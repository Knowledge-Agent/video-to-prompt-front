## 产品叙事与工作流升级

Video to Prompt v2.0 主要围绕公开站点的产品表达做了一次系统升级。

### v2.0 更新内容

- 首页重新聚焦到 prompt package 生成能力
- 博客内容围绕镜头语言、工作流设计与提示词 QA 展开
- 工具页重新定义为参考视频分析工作流
- 法务页与 docs 内容去模板化，改为产品真实描述
- 更新日志与公开文案全面对齐 Video to Prompt 品牌

### 这次升级为什么重要

一个工作流产品必须能清楚表达自己的价值。v2.0 让外部用户更容易理解：Video to Prompt 不是简单功能演示，而是一套真实可落地的生产系统。
