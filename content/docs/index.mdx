## What is Video to Prompt?

Video to Prompt is a workflow tool that turns one reference video into a structured prompt package.

Instead of manually guessing how to describe a clip, you upload the video and extract:

- a scene summary
- a master prompt
- a short prompt
- a negative prompt
- a keyword stack
- a shot list

The goal is simple: help creators, prompt engineers, and production teams convert good reference footage into reusable prompt assets.
