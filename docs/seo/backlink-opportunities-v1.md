# Backlink Opportunities v1

This is an execution-first prospect list for `videotopromptgenerator.com`.
All targets below are outreach opportunities, not confirmed placements.

## Qualification Rules

Before outreach, qualify each target with:

- topical relevance to AI video, creator tooling, prompt engineering, or growth workflows
- organic visibility and editorial quality
- realistic editorial path (resource page, guest post, tool list, case study, interview)
- non-spam outbound link behavior

## Link Building Lanes

- Lane A: Tool directories and product databases (quick wins)
- Lane B: AI and creator newsletters (high-quality mentions)
- Lane C: Tutorial and workflow publications (evergreen links)
- Lane D: Community and ecosystem pages (durable links)
- Lane E: Comparative and review pages (high-intent links)

- Week 1: qualify 50 prospects, prioritize top 20
